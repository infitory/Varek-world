# VAREK — The Bone World

> A worldbuilding compendium for a low dark fantasy setting.
> Pre-apocalypse. Everything written here is true as of the current year.

## About this compendium

This is a structured worldbuilding archive for the world of **Varek** — a dying continent in the tradition of *Mörk Borg*, where magic is rare and costly, institutions are slowly rotting, and something is quietly beginning in the margins that no one has yet connected.

The compendium is organised so it can grow. New entries can be added to existing sections. New sections can be created as the world expands. Each file is a self-contained document with clear headers and consistent metadata.

## How to extend this

When adding a new entry:

1. **Identify the right file.** A new village goes in the relevant kingdom file. A new cult goes in `faiths/buried-faiths.md`. A new creature goes in `margins/creatures.md`.
2. **Match the existing format.** Each entry has a header (`##`), optional metadata (type, status), and a body written in the same register as the rest of the compendium.
3. **Cross-reference.** If a new entry connects to existing ones, link them inline: `[Aldric](../characters/aldric.md)`, `[Ostmere](../kingdoms/ostmere.md)`.
4. **Preserve tone.** The compendium is written as if by a dry, slightly tired scholar. Short sentences. No hype. Facts first, implications second, judgment last or absent.

## Tone guide

The world of Varek is **low dark fantasy**. Keep the following in mind when writing or generating new material:

- **No heroes.** No chosen ones, no prophesied saviours. People are tired, compromised, and surviving.
- **No clean magic.** Magic is rare, expensive, mostly broken, and usually bad for the person using it.
- **No final victories.** The world is slowly dying. Some things can be delayed. Very few can be reversed.
- **Institutions fail slowly.** Kingdoms don't collapse dramatically; they rot from the inside while pretending to function.
- **The margins matter.** What is happening in the empty villages, the sealed mines, the crewless ships — these are more important than the wars of kings.
- **Faith is ambiguous.** The gods may or may not be real. When they answer, the answer is rarely a comfort.
- **Death is ordinary.** Characters die of disease, cold, poverty, and bad luck as often as they die of violence.
